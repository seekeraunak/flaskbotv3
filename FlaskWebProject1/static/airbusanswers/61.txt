1) AOC Datalink is an option and needs to be activated (enabled). This is done by Pin Programming, through Airbus (RFC/SB); this for Legacy FM. For Pegasus FM (Flight Management) this activation is done through OPC (Option Configuration File). Soft pin programming; still through Airbus RFC (Request For Change), Option activation. 2) Then Datalink function needs to be enabled in the Airline Policy File (APF) whis is part of Navigation Data Base for Legacy Aircrafts; in the AMI (Airline Modifiable Information) for the Pegasus FM.
